# **App Name**: Design Portfolio

## Core Features:

- Project Display: Display interior design projects in a grid layout with key details.
- Category Grouping: Group projects into categories such as Corporate, Residential, and Hospitality.
- Detailed Project Pages: Showcase individual projects with image galleries and descriptions.
- Contact Form: Allow potential clients to send messages directly through the site.
- Dynamic Content: Fetch all data to the application from a file hosted on the web.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a sophisticated, clean base.
- Background color: Light grey (#FAFAFA) for subtle contrast and a modern feel.
- Accent color: Muted gold (#BDBDBD) for highlights and call-to-actions.
- Headline font: 'Playfair' serif for headings to evoke elegance and classic design. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' sans-serif for body text, ensuring readability and a modern touch. Note: currently only Google Fonts are supported.
- Simple line icons for navigation and key actions.
- Full-width hero images, grid layouts for project thumbnails, and ample whitespace.
- Subtle fade-in effects for images and transitions between pages to enhance user experience.
